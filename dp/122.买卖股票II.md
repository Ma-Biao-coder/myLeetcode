122 买卖股票最佳时机
8 mai 2022
这道题允许多次操作，上一道题只允许操作一次

贪心算法
每一次拿第二天的与前一天的比较，如果前一天比第二天的低，则计算一次利润

```js
/**
 * @params number[] price
 * @return number
*/
var maxProfit = function(price){
    let maxProfit = 0
    //因为有和第二天的作比较，所以for只用遍历到倒数第二天
    for(let i = 0; i < price.length - 1; i++){
        if(price[i] < price[i + 1]){
            maxProfit += price[i+1] - price[i]
        }
    }
    return maxProfit
}
```
